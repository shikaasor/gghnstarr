Load `resources/tools/AMR Resource Repository.xlsx` with `data_only=True`, read Sheet1, skip the header, iterate the 172 data rows (columns A-H per 20-RESEARCH.md: Resource name, Type, Purpose, Description, Source, Accessible from, Region, Global use). Define a `build_items()` function returning a list of dicts matching EducationItem:
    - id: slugify(title) lowercased, non-alphanumeric collapsed to single hyphen, trimmed; prefix every imported id with `amr-` to guarantee no collision with the 15 curated ids; on duplicate slug append `-2`, `-3` (track seen ids in a set).
    - tab: 'training' if Type contains 'course' (case-insensitive) OR Purpose contains 'Training'; else 'resources'.
    - title: column A, stripped.
    - format: map column B Type via an explicit TYPE_TO_FORMAT table (per 20-RESEARCH.md): Journal Paper / Journal article / Article / Preprint paper / Paper / Conference Paper / Patent -> 'Publication'; Website / Mobile application -> 'Article'; Online course / Online Course -> 'Course'; Video -> 'Video'; all else (Toolkit, Document, Report*, Guide, Poster, Book, E-book, Book chapter, Report pending) -> 'Download'. Unmapped -> 'Download'.
    - topics: default ['Research'] for every imported record (decision 3 + Pitfall 3 — Purpose too messy for the 8-value enum). Do NOT parse Purpose into topics.
    - audiences: rule-based default by format (decision 3 inference) — 'Publication'/'Article'/'Course' -> ['Healthcare Worker']; 'Download' -> ['Policymaker']; 'Video' -> ['General Public']. Single-element arrays.
    - year: regex `(19|20)\d{2}` against Source (column E); take the last match if multiple; if no match or Source blank, OMIT the year key entirely (do not emit null/0).
    - source: column E stripped; if blank, derive host from url (urllib.parse.urlparse netloc, strip leading 'www.'); if still empty, 'Unknown'.
    - sourceVerified: false for all imported records (links not click-tested; per 20-RESEARCH.md recommendation).
    - url: column F stripped; if it does not start with 'http' (case-insensitive), wrap as `https://doi.org/<value>` (12 bare DOIs).
    - description: column D stripped (NEW field).
    - region: map column G via REGION_MAP to a WHORegion token: 'African Region (AFRO)'->'AFRO', 'European Region (EURO)'->'EURO', 'Region of the Americas (PAHO)'->'PAHO', 'Eastern Mediterranean Region (EMRO)'->'EMRO', 'Western Pacific Region (WPRO)'->'WPRO', 'South-East Asia Region (SEARO)'->'SEARO', 'All regions'->'All regions'. Unrecognized -> 'All regions' with a printed warning.
    In this task the script's `__main__` block must NOT write the file yet — only call build_items(), print `len(items)` (must be 172) and one sample dict for inspection. Trust openpyxl's decoded UTF-8 strings (Pitfall 1 — do not "fix" curly-quote/accent glyphs).
