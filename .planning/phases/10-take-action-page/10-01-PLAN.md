    - Type: `type CardId = 'pledge' | 'commitment'`
    - State: `expanded: CardId | null` init 'pledge' (locked decision: default Pledge pre-expanded)

    - State: `toast: { message: string } | null` init null
    - useEffect on mount: read window.location.hash, strip '#', if 'pledge' or 'commitment' setExpanded(hash)

    - Render: two-column card grid (`grid grid-cols-1 md:grid-cols-2 gap-6`)
    - Each card: white bg, rounded-lg, border border-slate-200, shadow-sm, p-6
    - Card header (always visible): icon + title + description + ChevronDown (rotates 180deg when expanded, hidden if locked)
    - Locked card shows a green CheckCircle badge instead of ChevronDown with "Submitted" text
    - Expand region: `overflow-hidden transition-all duration-300` with max-h-0 collapsed / max-h-[2000px] expanded (pattern from AccordionSection.tsx)
    - Each card has an `id` attribute matching its CardId ('pledge' or 'commitment') for scroll targeting
    - Pledge card: title "Sign the Public Pledge", description "Add your commitment to the fight against AMR across Africa.", icon = HandHeart or Heart from lucide-react; passes `onSuccess={() => handleSuccess('pledge', "Your pledge contributes to the fight against AMR in Africa.")}` to PledgeForm

    - Renders ActionToast when toast !== null; onDismiss sets toast to null
    - Section heading above grid: `<h2>` "Make Your Commitment" or similar, centered
