  <action>
    Create `app/components/briefs/BriefGrid.tsx` as a `'use client'` component. This component owns all filter state — it MUST NOT call getAllBriefs() or any fs-using function (Node.js only; will break client bundle).

    ```typescript
    'use client';
    import { useState, useMemo } from 'react';
    import type { Brief, Expert } from '@/lib/types';
    import BriefCard from './BriefCard';

    interface BriefGridProps {
      briefs: Brief[];
      experts: Expert[];
    }
    ```

    FILTER STATE: Two pieces of state — `monthFilter: string` and `themeFilter: string`, both initialized to `''` (empty = no filter applied).

    DERIVED DATA (via useMemo — never hardcode):
    - `months`: unique month labels derived from `brief.publicationDate`. Pattern: `new Date(b.publicationDate).toLocaleString('en-US', { month: 'long', year: 'numeric' })`. Deduplicate with a seen Set, preserve order of first appearance.
    - `themes`: unique themes from all `brief.themes` arrays. Use `briefs.flatMap(b => b.themes)` then `[...new Set(all)].sort()`. Use `b.themes.includes(themeFilter)` for matching — NOT equality — because briefs have multiple themes.

    FILTERING (via useMemo):
    ```typescript
    const filtered = useMemo(() => briefs.filter(b => {
      const monthMatch = !monthFilter ||
        new Date(b.publicationDate).toLocaleString('en-US', { month: 'long', year: 'numeric' }) === monthFilter;
      const themeMatch = !themeFilter || b.themes.includes(themeFilter);
      return monthMatch && themeMatch;
    }), [briefs, monthFilter, themeFilter]);
    ```

    RENDER STRUCTURE:
    - Filter bar (locked: top bar, two dropdowns, active filter shown in dropdown value itself):
      ```jsx
      <div className="flex flex-wrap gap-4 mb-8">
        <select value={monthFilter} onChange={e => setMonthFilter(e.target.value)}
                className="border border-slate-300 rounded px-3 py-2 text-sm text-slate-700 bg-white focus:outline-none focus:ring-2 focus:ring-teal-500">
          <option value="">Month</option>
          {months.map(m => <option key={m} value={m}>{m}</option>)}
        </select>
        <select value={themeFilter} onChange={e => setThemeFilter(e.target.value)}
                className="border border-slate-300 rounded px-3 py-2 text-sm text-slate-700 bg-white focus:outline-none focus:ring-2 focus:ring-teal-500">
          <option value="">Theme</option>
          {themes.map(t => <option key={t} value={t}>{t}</option>)}
        </select>
      </div>
      ```
    - Empty state (locked: message + Clear filters button):
      ```jsx
      {filtered.length === 0 ? (
        <div className="text-center py-16">
          <p className="text-slate-600 mb-4">No briefs match your current filters.</p>
          <button
            onClick={() => { setMonthFilter(''); setThemeFilter(''); }}
            className="text-teal-600 underline hover:text-teal-500 text-sm">
            Clear filters
          </button>
        </div>
      ) : (
        <div className="grid grid-cols-1 md:grid-cols-2 lg:grid-cols-3 gap-6">
          {filtered.map(b => (
            <BriefCard
              key={b.slug}
              brief={b}
              expert={experts.find(e => e.id === b.authorId)}
            />
          ))}
        </div>
      )}
      ```
    - Responsive grid (discretion: 1 col mobile, 2 col md, 3 col lg) is covered by the className above.
    - Wrap everything in a React fragment or a single div with no extra Container — the page.tsx will handle Container wrapping.
