## Problem

The briefs library doubles as both the awareness hub and education library, but is not structured or branded as either. There is no infographic gallery, no short explainer articles, and no way to filter content by target audience (general public vs. healthcare workers vs. policymakers).

## Solution

Create dedicated /awareness page with infographic gallery and short explainer articles. Add audience filter (Public / HCW / Policymaker) across all resource pages. Tag existing briefs and new content by audience type.
