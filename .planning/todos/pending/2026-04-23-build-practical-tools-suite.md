## Problem

The platform spec requires stewardship checklists, self-assessment quizzes, and facility reporting templates. None of these exist. Healthcare workers visiting the site have no actionable tools to take back to their facilities.

## Solution

Build /tools page with: interactive stewardship checklist builder (printable output), self-assessment quiz flows with scored results, downloadable facility reporting templates (PDF or DOCX). Keep client-side only — no backend needed for static export.
