
    State: `const [email, setEmail] = useState('')` and `const [state, setState] = useState<FormState>('idle')`

    Submit function — CRITICAL CORS WORKAROUND: Google Apps Script Web Apps cannot respond to CORS preflight (OPTIONS) requests. Sending Content-Type: application/json triggers a preflight that silently fails. The required pattern is:
    ```typescript
    async function submitNewsletter(emailValue: string): Promise<'success' | 'error'> {
      try {
        const res = await fetch(process.env.NEXT_PUBLIC_GAS_URL!, {
          method: 'POST',
          redirect: 'follow',      // GAS /exec returns 302 redirect — must follow it
          headers: { 'Content-Type': 'text/plain;charset=utf-8' },  // Simple request — no preflight
          body: JSON.stringify({ email: emailValue, source: 'homepage', timestamp: new Date().toISOString() }),
        });
        if (!res.ok) return 'error';
        const json = await res.json();
        return json.status === 'success' ? 'success' : 'error';
      } catch {
        return 'error';
      }
    }
    ```

    handleSubmit: `e.preventDefault(); setState('submitting'); const result = await submitNewsletter(email); setState(result);`

    Success state (render when state === 'success') — replaces entire section:
    ```tsx
    <section className="bg-slate-100 py-16 text-center">
      <Container>
        <CheckCircle className="mx-auto text-teal-600 mb-3" size={32} />
        <p className="text-navy-950 font-medium text-lg">You're subscribed.</p>
        <p className="text-slate-600 mt-1">We'll send each policy brief directly to your inbox.</p>
      </Container>
    </section>
    ```

    Default form state:
    - Outer section: `className="bg-slate-100 py-16"`
    - Container with `className="max-w-lg text-center"` — pass className prop to Container
    - Heading: `<h2 className="font-serif text-2xl text-navy-950 mb-2">Stay Informed</h2>`
    - Sub: `<p className="text-slate-600 mb-6">Receive each policy brief as it's published.</p>`
    - Form: `<form onSubmit={handleSubmit} className="flex flex-col sm:flex-row gap-3">`
      - Input: `type="email" required value={email} onChange={(e) => setEmail(e.target.value)} placeholder="your@email.com"` with class `"flex-1 border border-slate-200 rounded px-4 py-2 text-navy-950 focus:outline-none focus:ring-2 focus:ring-teal-600"`
      - Button: `type="submit" disabled={state === 'submitting'}` with class `"bg-teal-600 hover:bg-teal-500 disabled:opacity-60 text-white font-medium px-6 py-2 rounded transition-colors inline-flex items-center gap-2"`.
